# Simple ACL RBAC Sistem use Nodejs and MongoDB


## Dependencies used

    accesscontrol 
    bcrypt 
    body-parser
    dotenv 
    express 
    jsonwebtoken 
    mongoose 



## Routes endpoint

    post('/signup')
    post('/login')
    get('/user/:userId')
    get('/users')
    put('/user/:userId')
    delete('/user/:userId')


    


## Init project

    npm install 
    cd /server
    npm server
