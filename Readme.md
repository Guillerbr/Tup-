# Simple ACL RBAC Sistem use Nodejs and MongoDB


## Dependencies used

    accesscontrol 
    bcrypt 
    body-parser
    dotenv 
    express 
    jsonwebtoken 
    mongoose 



## Routes endpoint

    post('/signup')
    post('/login')

    get('/user/:userId')
    get('/users')
    put('/user/:userId')
    delete('/user/:userId')

    get('/balances')
    post('/balance')
    put('/balance/:balanceId')
    delete('/balance/:balanceId')



    
## Init project

    npm install 
    cd /server
    npm server
